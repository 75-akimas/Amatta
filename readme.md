# Amatta!

公開URL: http://ec2-18-224-179-78.us-east-2.compute.amazonaws.com/
- Cookpad 2019 internship 10daysコースにて作成したモバイルウェブアプリケーションです。
- バックエンドはLaravelのWebAPIで、フロントはVuejsを用いたSPAです。

### To build
```
.envを作成しデータベースを作成した後、

npm install

npm run deployment

php artisan update

php artisan migrate

php artisan serv
```
